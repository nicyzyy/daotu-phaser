=== USAGE ===
1. Extract frames from sprite sheets using the code in readme_animations.md
2. Apply green (#00FF00) chroma key to remove background
3. Import into Phaser/Unity/Godot using frame dimensions:
   - Idle: 1024/4 = 256px per frame
   - Walk: 1024/8 = 128px per frame
   - Ride Idle: 1024/4 = 256px per frame
   - Ride Move: 1024/8 = 128px per frame
   - Mount/Dismount: 1024/6 = ~170px per frame
4. Set anchor point to center-bottom (0.5, 1.0)
5. Configure frame rates per animation type (see readme)

=== NOTES ===

- Sprites are generated with top-down RPG perspective (~30° elevation)

- For production: consider manual cleanup or skeletal animation for pixel-perfect consistency
